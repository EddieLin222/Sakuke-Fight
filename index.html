Intro.

( 火影忍者打鬥遊戲 ) ，透過 Phaser.js 提供的物理碰撞 ，精靈圖播放 ，快速引用遊戲所需的必要需求。

Step1.
準備遊戲所需用到的素材，如 : 雪碧圖 (Sprite sheet)，用於製作遊戲動畫。
Step2.
引入 Phaser.js，將官方的 Getting Started Code 複製貼進自己的 HTML，成功的話打開網站有一個酷炫的動畫出現。
Step3.
將素材替換成自己想做的遊戲內容及製作符合遊戲的事件，Phaser在網路上的教學資源很多，找到遊戲模式相近或者動畫邏輯相同的例子。
Step4.
參考官方的文檔 ( 主要為 Phaser API Documentation )，逐步加入JS的判斷式，漸漸感受到與自己理想中的操作與畫面一致。

Rule.

基本遊戲規則介紹。

遊戲方式 | Rules Of The Game |
『 基本規則 』
玩家及對手一開始的生命 Health Point（HP）都為100點
玩家可使用鍵盤操控角色，每成功攻擊到對手一次可以使對手的生命下降10點（絕招攻擊成功可使對手生命下降50點）
而當玩家被對手攻擊到也會下降10點的生命
『 敵人追擊 』
玩家與敵人距離在一定範圍內才會被敵人追擊，可在此範圍外使用集氣招式攻擊
